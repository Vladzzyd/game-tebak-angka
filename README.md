# Game Tebak Angka

Game CLI Python interaktif di mana pemain menebak angka antara 0-100. Program memberikan petunjuk apakah tebakan terlalu tinggi atau terlalu rendah, dan menghitung total tebakan.

## Fitur

* Tebak angka dengan hint naik (⬆️) atau turun (⬇️)
* Total tebakan dihitung
* Bisa dimainkan berulang kali
* Warna terminal untuk meningkatkan interaksi

## Requirement

* Python 3.x
* Terminal yang mendukung ANSI escape color (untuk warna)

## Warna yang Digunakan

* Merah (RED): Error atau input salah
* Hijau (GREEN): Jawaban benar
* Kuning (YELLOW): Petunjuk naik/turun
* Cyan (CYAN): Border dan kotak interaktif
